📚 Book Recommendation Chatbot (NLP + Deep Learning)

An intelligent Book Recommendation Chatbot built using Natural Language Processing (NLP) and Deep Learning.
The chatbot understands user queries and recommends books based on the detected intent.

This project demonstrates the full pipeline of building an AI system including:
1- Text preprocessing
2- Feature engineering (Bag of Words)
3- Deep Learning model training
4- Model evaluation and visualization
5- Model deployment preparation
